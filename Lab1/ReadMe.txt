
The PDF contains explanation of this lab and the tasks which have to be carried out.

The test file contains the main source code and sample.txt contains the test input
Make sure you place the "sample.txt" and and "test.java" in same folder
Now perform the following steps to test the code.

Steps to see output
1. compile the code using command: "javac test.java"
2. To run for FCFS type in command: "java test sample.txt FCFS"
3. To run for SJF type in command: "java test sample.txt SJF"
4. to run for RR type in command: "java test sample.txt RR <time-quantum>" (for <time-quantum> you need to specify time in integer)
